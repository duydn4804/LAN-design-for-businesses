# LAN-design-for-businesses
Design LAN for businesses using Cisco Packet Tracer Project description: The project implements LAN design and simulation for a business with 4 buildings (Building A, B, C, D) including departments: Engineering, Security, Accounting, Human Resources, Board of Directors and Central Operations. Use Cisco Packet Tracer to:
 - Divide the network using VLAN and subnetting.
 - Dynamic IP configuration using DHCP Server.
 - Wired and wireless network connection (Wi-Fi).
 - Apply OSPF routing protocol to optimize transmission lines.
